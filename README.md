# Image & Text Replacer Chrome Extension

Webサイト上の **すべての画像をランダムな画像に置換**し、  
同時に **サイト内のすべての文字をランダムな文字列に置換**する  
Chrome拡張機能です。

※ YouTube（youtube.com）では自動的に無効化されます。

---

## 機能

- サイト内の `<img>` タグをすべてランダム画像に置換
- サイト内の表示テキストをすべてランダム文字列に置換
- 動的に読み込まれる要素（無限スクロール等）にも対応
- 拡張機能アイコンで ON / OFF 切り替え可能
- YouTube は完全除外（誤動作防止）

---

## 動作環境

- Google Chrome（最新版推奨）
- Chromium系ブラウザ（Edge など）でも動作する可能性あり

---

## ダウンロード方法

### 方法①：GitHubからZIPでダウンロード
1. このリポジトリのページを開く
2. 右上の **「Code」** ボタンをクリック
3. **「Download ZIP」** を選択
4. ZIPファイルを任意の場所に解凍する

## Chromeへの導入方法（開発者モード）

この拡張機能は Chrome Web Store を通さず、  
**「パッケージ化されていない拡張機能」**として読み込みます。

### 手順

1. Google Chrome を起動する
2. アドレスバーに以下を入力して開く  
chrome://extensions/
3. 画面右上の **「デベロッパーモード」** を ON にする  
4. 左上に表示される  
**「パッケージ化されていない拡張機能を読み込む」** をクリック
5. このリポジトリをダウンロード・解凍したフォルダ  
（`manifest.json` があるフォルダ）を選択
6. 拡張機能が一覧に表示されたら導入完了

---

### 導入できない場合のチェックポイント

- `manifest.json` がフォルダ直下に存在しているか
- ZIPを解凍せずに読み込もうとしていないか
- Chromeのバージョンが古すぎないか

---

### 拡張機能の使い方

- Chrome右上の拡張機能アイコンをクリックすると ON / OFF 切り替え
- 状態は自動保存され、ページ再読み込みで反映されます
- YouTube（youtube.com）では自動的に無効化されます
